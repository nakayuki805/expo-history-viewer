# 万博予約履歴ビューアー

React（TypeScript）と Tailwind CSS で構築した単一ページのビューアーです。大阪・関西万博のマイチケットで取得できるチケット一覧 API の JSON を読み込み、入場予約およびパビリオン予約を見やすく整理して表示します。すべての処理はブラウザ内で完結するため、貼り付けたデータが外部に送信されることはありません。

## 主な機能
- JSON の手動貼り付けとファイル読み込み（`.json`/`.html`/`.webarchive`）に対応
- 入場予約とパビリオン予約をカード形式で一覧表示
- 予約日時、ゲート、利用状況などの主要項目をタグで表示
- 取得した画像パスを自動で公式サイトの完全 URL に変換

## 開発環境のセットアップ
```bash
npm install
npm run dev
```
`npm run dev` を実行すると、デフォルトで `http://localhost:5173` で開発サーバーが立ち上がります。

## ビルド
```bash
npm run build
```
生成された静的ファイルは `dist/` ディレクトリに出力されます。`dist` 以下を静的配信すればアプリをホスティングできます。ローカルで確認したい場合は以下を実行してください。

```bash
npm run preview
```

## JSON の用意方法
1. [マイチケット](https://ticket.expo2025.or.jp/myticket/) にログインします。
2. チケット一覧 API (`https://ticket.expo2025.or.jp/api/d/my/tickets/`) をブラウザで直接開きます。
3. 表示された JSON をコピーするか、ファイルとして保存します（スマートフォンでは Web Archive でも可）。
4. 本アプリで JSON を貼り付ける、またはファイルを読み込んで結果を確認します。

## 注意事項
- 本ツールは非公式の個人向けビューアーです。利用は自己責任でお願いします。
- 解析に失敗する場合は、ブラウザから取得したままの JSON であるかを再確認してください。
- 読み込んだデータはブラウザにのみ保持され、アプリを閉じると消去されます。
